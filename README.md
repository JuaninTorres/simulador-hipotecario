simulador-hipotecario
=====================

Simulador de un Crédito Hipotecario, se trabaja con un archivo php que captura el valor de la UF desde la página del Servicio de Impuestos Internos (www.sii.cl).

Para el funcionamiento correcto de la clase, es necesario tener activada la expencion de php "curl", la cual se puede instalar en las máquinas con debian con
``` bash
sudo apt-get install php5-curl
```
